# Enunciado

Una fábrica de carros requiere un sistema que controle la fabricación de carros en sus diferentes plantas.

Se debe considerar únicamente como **partes de cada carro**:

- **Llantas**
- **Chasis**

## Información de las Llantas

Para cada llanta es importante registrar:

- **Tamaño**
- **Tipo**

## Información del Chasis

Para cada chasis se debe considerar:

- **Peso**
- **Material**

El material del chasis puede ser:


- **Aluminio**
- **Acero**

## Información del Carro

De cada carro es necesario conocer:

- **Qué llantas tiene**
- **Qué chasis tiene**
- **Color del carro**

## Información de la Planta

De cada planta es importante conocer:

- El **tamaño de las llantas** que fabrica
- El **tipo de llantas** que fabrica
- El **peso del chasis** que suele fabricar
- El **material del chasis** que utiliza
- Los **colores de carros** que suele producir

Además, cada **planta está en la capacidad de fabricar carros**.

## Ejemplo de uso del sistema

Se debe implementar el siguiente escenario:

- Crear **dos plantas**.
- Cada planta debe **fabricar 100 carros**.
- El programa debe **mostrar en pantalla la información de cada nuevo carro fabricado**.
